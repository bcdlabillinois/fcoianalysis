# fcoianalysis
Basic code for different fixed array and fcoi analyses of infant fNIRS data reported in the following publication:

Liu, Y., Sanchez Hernandez, F., Ting, F., & Hyde, D. C. (2022). Comparing fixed-array and functionally-defined channel of interest approaches to infant functional near-infrared spectroscopy data. NeuroImage, 261, 119520. https://doi.org/10.1016/j.neuroimage.2022.119520
